## ‚úÖ Requirements Checklist

### Core Requirements (Must Have)
- [ ] Project runs without critical errors
- [ ] At least 2-3 custom classes demonstrating OOP
- [ ] Code demonstrates encapsulation (private variables, methods)
- [ ] Uses at least ONE external library/framework
- [ ] Code is well-commented (explains WHY, not just WHAT)
- [ ] Clear, meaningful variable and function names
- [ ] README.md with project description and setup instructions
- [ ] Can demonstrate project working in presentation

### OOP Concepts Demonstrated (Flexible Based on Language)
- [ ] Classes/Objects used appropriately
- [ ] Methods organized logically
- [ ] Data encapsulation (protecting data)
- [ ] If Java: Show inheritance, polymorphism where it makes sense
- [ ] If other language: Show equivalent OOP structure

### Code Quality Standards
- [ ] Consistent indentation and formatting
- [ ] No major dead code or commented-out blocks
- [ ] Functions/methods are reasonable size
- [ ] DRY principle applied (Don't Repeat Yourself)
- [ ] Edge cases considered
- [ ] Input validation where needed
